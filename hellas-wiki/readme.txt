Complete WordPress-native wiki experience for the Hellas Region project. Automatically syncs data from the HellasForms repositories and renders fully linked Pokédex entries, items, moves, abilities, locations, and guides.

== Description ==

* Registers custom post types for species, forms, moves, abilities, items, locations, and guides.
* Imports JSON data from the HellasForms GitHub repository via webhook or manual uploads.
* Offers an interactive admin wizard, importer, and queue to streamline content curation.
* Provides responsive templates with Hellas Forge dark styling.
* Supports automatic tooltip summaries and rarity tiering.

== Installation ==

1. Upload the `hellas-wiki` folder to `/wp-content/plugins/`.
2. Activate the plugin through the "Plugins" screen.
3. Visit **Hellas Wiki → Settings** to configure GitHub repository access and webhook secrets.
